# RaftLab Food Ordering System

A full-stack food ordering and order tracking application built using React, Node.js, Express, and MongoDB.

---

# Tech Stack

## Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Testing
- Jest
- Supertest
- Vitest
- React Testing Library

---

# Features

- Browse food menu
- Add/remove cart items
- Place orders
- Order tracking system
- Dynamic order status updates
- Responsive UI
- API integration
- Form validation
- Backend API testing
- Frontend component testing
